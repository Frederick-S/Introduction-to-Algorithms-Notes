# 4.1 The maximum-subarray problem
## 4.1-1
It returns the index and value of the biggest negative number in A.

## 4.1-2
```
FIND-MAXIMUM-SUBARRAY-BRUTE-FORCE(A)

max = 0
n = A.length
low = -1, high = -1

for i = 1 to n
    sum = 0

    for j = i to n
        sum += A[j]

        if sum > max
            max = sum
            low = i
            high = j

return (low, high, max)
```
