# 10.2 Linked lists
## 10.2-1
We can implement `INSERT` in O(1) time, but cannot implement `DELETE` in O(1) time, it's O(n).

## 10.2-2
```
PUSH(x)

L.insert(x)
```

```
POP()

if L.head == NIL
    error "underflow"
x = L.head
L.head = L.head.next
return x
```

## 10.2-3
```
ENQUEUE(x)

new = SinglyLinkedNode(x)
if L.head == NIL
    L.head = new
    L.tail = new
else
    L.tail.next = new
    L.tail = new
```

```
DEQUEUE()

if L.head == NIL
    error "underflow"
x = L.head
L.head = L.head.next
return x
```
